# Contributing to Hardware Switzerland

## Ways to Contribute

### 1. Technical Implementation
- FPGA/ASIC designs for PCIe/SATA monitoring
- Protocol decoders for different interfaces
- Low-level timing analysis

### 2. Security Research
- Attack pattern signatures
- Anomaly detection algorithms
- Threat model development

### 3. Documentation & Outreach
- Use case documentation
- Compliance analysis
- Community building

## Getting Started
1. Fork the repository
2. Check Issues for "good first issue" labels
3. Join discussion in Issues before major work

## Communication
- GitHub Issues for technical discussion
- No private channels - everything public
